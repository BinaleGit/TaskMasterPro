About This App

Welcome to the AI-powered To-do List, an intelligent task management app designed to help you prioritize and manage your tasks efficiently.

This app uses machine learning algorithms to assign priorities to tasks based on deadlines and estimated completion times. It helps you focus on the most urgent and important tasks first by automatically sorting tasks into categories such as Past Due, Today, Tomorrow, and Upcoming.

Key Features

  • Smart priority prediction using AI.
  • Automatic grouping of tasks by deadlines.
  • User-friendly interface with color-coded task lists for better visual organization.
  • Easy addition and management of tasks.

How It Works

The app uses a machine learning model that takes into account the estimated time required for a task and the task's deadline. Based on these inputs and your custom priority, the app calculates a final priority and orders the tasks accordingly.

The app is constantly learning and can be retrained as more tasks are added, ensuring that the task prioritization improves over time.

Technologies Used

  • Flask for the backend web server.
  • Pandas and Scikit-learn for machine learning and data management.
  • Tailwind CSS for a modern and responsive user interface.
